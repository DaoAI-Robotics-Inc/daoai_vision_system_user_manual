脚本如何运行
----------------------------

 - Hans示教器需要先点击“运行”、“使能”使机器人上电。

    .. image:: images/使能.png
        :scale: 80%

 - 接下来我们选择我们需要运行的程序：Hans示教器需要先点击“编程”、选择"main"程序、接着"call"不同的子程序、在下图第4步时选择子程序、点击第5步”运行“即可。

    .. image:: images/选择程序.png
        :scale: 80%
