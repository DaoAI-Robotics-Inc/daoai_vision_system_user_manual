PC预算应首先用于表中的组件。如果预算较高，您的支出大致应该按这个顺序完成：


- 将GPU升级到1070或类似版本

- 将CPU升级到i7-11700k或类似版本

- 将GPU升级到 RTX 2080/RTX 3080

- 将CPU升级到i9

- 将内存（RAM）升级到 64 GB


所有DaoAI 相机系列均支持所有的工业PC产品。
