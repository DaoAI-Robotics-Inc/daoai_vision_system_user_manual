视觉项目
============

本章将会详细介绍DaoAI机器人视觉认知系统的视觉项目，从创建一个项目开始。 

.. image:: images/vision_project.png

创建一个新项目
-------------------

启动服务器后，访问 `http://localhost:3000/ <http://localhost:3000/>`_ 进入DaoAI机器人视觉认知系统欢迎界面。

点击继续后，进入项目管理页面。

.. image:: images/vision_project_home.png


在这里点击，创建一个新项目
